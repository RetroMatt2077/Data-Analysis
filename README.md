🎮 Game Stats Analysis (Capstone Project)

A full game performance analytics system built using simulated player data.

🔍 Key Features

Custom SkillScore performance metric

Player leaderboard generation

Win rate vs accuracy analysis

Reaction time performance impact

Correlation analysis of gameplay stats


📊 Insights

Faster reaction times correlate with higher skill scores

Accuracy is a strong predictor of overall performance

Balanced players outperform high-specialization builds



---

⚔️ RPG Character Analysis

A fantasy-style dataset simulating RPG character stats and balancing systems.

🔍 Key Features

Class-based stat analysis (Warrior, Mage, Rogue, etc.)

PowerScore ranking system

Gold economy analysis

Character balance evaluation


📊 Insights

Certain classes dominate in raw power metrics

Speed-heavy builds are underrepresented but highly effective

Economy distribution is heavily skewed toward top performers



---

🌦️ Weather Data Analysis

A time-series analysis project simulating real-world weather patterns.

🔍 Key Features

Temperature trend analysis

Rolling averages (smoothing technique)

Correlation between weather variables

Monthly aggregation


📊 Insights

Temperature and humidity show weak correlation

Rolling averages reveal hidden trends in noisy data

Rainfall is highly irregular and event-driven
